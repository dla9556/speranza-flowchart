창고 — 아이템 보관 창고
  • 시작 용량 64 슬롯 → 최대 280+슬롯
  • 슬롯 확장 코인 구매
    인벤토리 화면에서 코인으로 슬롯 직접 구매 가능
    첫 구매: +24슬롯 / 5,000 코인 (이후 단계별 비용 증가)
  • 영구 확장 원정 보상
    원정 완료 시 +12슬롯 영구 지급 (초기화 후에도 유지)
  • 스택 병합 — 분산 아이템 병합 · 공간 절약
  • 자동 전시 — 고가 아이템 → 레이더 은신처 진열대 표시
  • 원정 창고 가치
    창고 가치 + 코인 합산 → 원정 목표치
  • 기록장 — 탐사 중 수집한 문서 · 로어 노트 보관
    발견한 지표면 메모 · NPC 통신 기록 · 세계관 자료 열람
